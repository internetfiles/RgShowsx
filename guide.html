Features

Shortcut Keys

Alt + 1 keys to go to home page quickly.

Alt + 2 keys to go to Movies page quickly.

Alt + 3 keys to go to Shows page quickly.

Alt + 4 keys to go to Anime page quickly.

Alt + 5 keys to go to LiveTV page quickly.

Alt + 6 keys to go to Settings page quickly.

Alt + T or t keys to go to Torrent page quickly.

Alt + M or m keys to watch a random movie.

Alt + S or s keys to watch a random series/show.

Alt + X or x keys to close the movies/series/shows player.

Alt + Delete keys to clear all the data.
